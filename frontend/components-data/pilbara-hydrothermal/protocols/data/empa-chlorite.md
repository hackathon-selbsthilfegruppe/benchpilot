**Method:** EMPA spot analyses on chlorite + serpentine grains; standard block including a Mg-Fe-Si chlorite reference (Mg# ~50); element maps over the cleanest chlorite zone; T extraction using the Vidal et al. 2006 calibration framework.

**Primary source:** Vidal et al. 2006 (see literature). protocols.io has a small number of analogue procedures — query terms: *"EMPA chlorite thermobarometry"* / *"electron microprobe chlorite"*.

**Relevance:** the methodological backbone for the deciding analysis (analyses → empa-thermobarometry). Aligned with the standards check open as task in the instruments component.
